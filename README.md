# Aurafy

Aurafy is a social influence platform where users "hype up" influencers, increasing their visibility by boosting a glowing aura. Influencers can open **vortexes** (channels) that grow brighter through user interaction and become more visible on a global map.

## Key Features

- **Aura farming** – help your favorite creators by steadily building their aura
- **Hype Me Up** – quickly summon your followers to boost your visibility
- **Visibility toggle** – choose whether your aura appears on the global map
- **Aura map** – explore the world to find the brightest vortexes

## Tech Stack

- **SwiftUI** for building native iOS interfaces
- **Firebase** or **Supabase** for real-time data and authentication
- **MapKit** to display aura locations on an interactive map
- Additional frameworks as the project grows

## Future Roadmap

- Vortex moderation features for influencers
- Aura reward system for active participants
- Global leaderboard showcasing top creators

## Getting Started

1. Clone this repository and open it in **Xcode**.
2. Select the `Aurafy` target.
3. Build and run on a device or simulator running **iOS 16** or later.
